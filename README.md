# 🍽️ Sistema de Restaurante

Sistema de restaurante desenvolvido em Python com FastAPI e Flet.

Permite criar e excluir itens do cardápio (admin), fazer pedidos (cliente) e gerenciar pedidos na cozinha. Os dados são salvos automaticamente em arquivo JSON.

Projeto focado em aprendizado de API, interface e fluxo real de sistema."# sistema-restaurante"
